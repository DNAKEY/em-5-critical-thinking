# EM 5.0 Critical Thinking Project Initiative

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/CHL987/em-5-critical-thinking)

A structured civic reasoning platform built on **Eureka Multiverse (EM) 5.0**, developed in collaboration with **Dr. Arturo López-Levy** and **Dr. Rafael Espín-Andrade**.

---

## Overview

This initiative pilots EM 5.0 as a structured critical thinking engine for political, social, and economic analysis across the Americas. It positions the platform as a civic reasoning instrument — making analysis **transparent, explainable, and reproducible**.

The project is grounded in Hannah Arendt's argument that unreflective thinking enables harm, and uses the **Wide Knowledge Discovery (WKD)** framework alongside **Compensatory Fuzzy Logic** to transform complex public questions into structured, auditable reasoning outputs.

---

## EM 5.0 Pipeline

| Role | Function |
|------|----------|
| **Core** | Classifies audience questions and orchestrates analytical routing |
| **Structurer** | Validates the problem, researches actors and context |
| **Descriptor** | Models variables, institutions, and relationships |
| **Predictor** | Builds scenarios, likelihoods, and uncertainty bands |
| **Prescriptor** | Ranks alternatives using compensatory reasoning |
| **Actioner** | Translates recommendations into implementable steps |
| **Publisher** | Converts validated analysis into scripts, reports, and public summaries |

## WKD Language Layers

The initiative ensures outputs are legible across all expertise levels through five layers:

- **Natural** — Plain-language accessible analysis
- **Semantic** — Structured framing and conceptual maps
- **Graphical** — Visual relationship diagrams
- **Algebraic** — Quantitative indicators and models
- **Symbolic** — Formal logical predicates

## Deploy on Vercel

### One-Click Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/CHL987/em-5-critical-thinking)

### Manual Deploy via CLI

```bash
# Install Vercel CLI
npm install -g vercel

# Clone and enter repository
git clone https://github.com/CHL987/em-5-critical-thinking.git
cd em-5-critical-thinking

# Deploy to production
vercel --prod
```

### Import via Vercel Dashboard

1. Go to [vercel.com/new](https://vercel.com/new)
2. Select **Import Git Repository**
3. Choose `CHL987/em-5-critical-thinking`
4. Framework Preset: **Other** (static site)
5. Click **Deploy**

> No build step required. Vercel detects `index.html` at the root and deploys instantly.
